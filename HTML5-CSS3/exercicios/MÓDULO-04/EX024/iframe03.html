NAVEGAÇÃO NO IFRAME

Aqui vemos como é possível navegar abrindo vários locais dentro do mesmo iframe e para isso foi criada uma lista com links internos e externos que quando ativados irão abrir dentro do quadro.

  • Curso em Vídeo - Página externa
  • História da internet-pdf - Arquivo-interno
  • Foto - Página interna
  • GitHub - Página externa


A maioria das páginas externas não permitem ser colocadas dentro de iframes e recusam a conexão como o link para o GitHub dado como exemplo.
