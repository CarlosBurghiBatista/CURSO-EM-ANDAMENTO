Revisando o primeiro exercício do curso

Nesse exercício foi aprendido o uso das tags "h1" e "p"

A tag "h1" é utilizada normalmente para títulos no corpo do documento, pode ser utilizada várias vezes no mesmo documento se for necessário.É o maior tamanho para títulos, mas existem outros 5 tamanhos que serão vistos mais adiante.

A tag "p" é utilizada para escrever parágrafos dentro do texto no documento. Normalmente para descrição relacionada com um título colocado anteriormente.
