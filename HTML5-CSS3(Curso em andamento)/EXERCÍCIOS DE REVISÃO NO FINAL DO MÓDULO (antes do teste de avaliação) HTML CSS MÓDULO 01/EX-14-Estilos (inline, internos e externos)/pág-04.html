Estilo Externo

O Estilo Externo consiste em um arquivo CSS inserido pela tag <link>assim como as configurações internas, na área da cabeça do Documento (head).

Dessa forma é possível aplicar todas as configurações contidas nesse arquivo CSS à todas as páginas existentes no site ou às quais se desejar.Bastando para isso adicionar o mesmo arquivo CSS no documento de cada uma delas.

Visualmente na página, o resultado ficaria igual ao exemplo do Estilo Interno, portanto não daremos exemplo aqui, mas devemos observar com atenção o desenvolvimento do Documento no Editor de texto (VScode recomendado).
